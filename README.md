# 📊 Retail Sales Dashboard | Power BI

## 📌 Project Overview
This project presents an interactive Retail Sales Dashboard developed in Microsoft Power BI to analyze business performance through KPIs and visual insights.

## 🚀 Features
- Revenue & Profit Analysis
- Monthly Sales Trend
- Sales by Category
- Top 10 Products by Revenue
- Sales by Region
- Top Customers Analysis
- Payment Method Distribution
- Interactive Filters

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

## 📈 Key Insights
- Revenue increased steadily throughout the year.
- Electronics generated the highest sales.
- West region contributed the highest revenue.
- Credit Card was the most preferred payment method.
